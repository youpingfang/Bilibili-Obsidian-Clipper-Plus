# Bilibili Obsidian Clipper Plus v1.0.14 发布说明

本次更新主要修复了「稍后再看」页面抓字幕不稳定的问题，并补充了一些导出与兼容性优化。

## 更新内容

### 1. 优化稍后再看页面的字幕抓取

- 现在支持更多 `watchlater` 页面链接形态
- 修复同一个视频在“稍后再看”里一会能抓、一会不能抓的问题
- 当页面跳转或内容脚本未及时注入时，会自动重试一次，提升成功率

### 2. 优化导出文件名设置

- 新增“文件名前包含导出日期”开关
- 可按需决定导出的 Markdown 文件名是否带日期前缀
- 多分P视频导出时，文件名会自动带上 `P序号 + 分P标题`

### 3. 优化错误提示

- 将部分浏览器通信报错替换为更易理解的中文提示
- 当页面不支持抓取或需要刷新时，提示会更清晰

### 4. 优化浏览器打包兼容性

- 调整 Chrome / Firefox 打包产物
- 修复 Firefox 临时加载附加组件时的后台脚本兼容问题

## 修复问题

- 修复 `https://www.bilibili.com/list/watchlater` 与 `https://www.bilibili.com/list/watchlater/` 两种链接表现不一致的问题
- 修复部分 `watchlater` 页面首次打开后无法立即抓取字幕的问题
- 修复导出文件名规则不够灵活的问题

## 说明

- 仍然只支持“本身存在字幕轨”的 B 站视频
- 如果某个视频没有字幕轨，插件仍然无法抓取字幕
- 更新后如果遇到异常，建议先重新加载扩展，再刷新视频页面重试
